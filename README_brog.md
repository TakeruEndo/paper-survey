## jekyll + Github Pages

```
# 2.5.0だとうまくローカルサーバーを立ち上げられなかった
$ rbenv install 2.4.0
# ver確認
$ rbenv versions
$ rvenv global 2.4.0
$ mkdir ~/.rbenv/versions/paper-survey
$ ruby-build 2.4.0 ~/.rbenv/versions/paper-survey

$ bundle install
$ gem install gekyll
$ jekyll serve

```
